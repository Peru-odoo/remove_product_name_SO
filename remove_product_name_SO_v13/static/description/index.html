Remove Product Name SO

Remove Product Name SO is a plugin to remove the product name automatically displayed in the product description in the sale order lines. If you are tired of removing the product name from the description lines of your quotes, this module becomes absolutely vital for you. Remove Product Name SO will take care of that for you. Indeed, when you create a quote, if the product has a description, then it will be automatically added in the description box without the product name. If however you have not defined a description to a specific product, then the description of it will be filled only with the name.
